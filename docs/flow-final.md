# The full flow


This is flow represents the final application this workshop creates. You can import
it into your Node-RED editor using the Import option from the main menu.

You will need to have installed the extra node modules used by this flow:

 - `node-red-dashboard`
 - `node-red-node-ui-table`
 - `node-red-node-ui-webcam`
 - `node-red-contrib-tfjs-coco-ssd`
 - `node-red-node-annotate-image`
